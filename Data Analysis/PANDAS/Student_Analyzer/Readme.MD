# 📊 Student Performance Analyzer

A simple CLI-based Student Performance Analyzer built using **Python** and **Pandas**.

This project demonstrates how to load, analyze, and manipulate CSV data using Pandas.

---

## 🚀 Features

- View complete dataset
- Show dataset shape
- Display column names
- Show first and last 5 records
- Display dataset information
- Show statistical summary
- Calculate total marks
- Calculate average marks
- Assign grades (A/B/C/D/F)
- Find topper
- Find lowest scorer
- Show students scoring above 80%
- Sort students by average marks
- Export updated dataset to a new CSV file

## ▶️ How to Run

```bash

```

Go to the project folder

```bash
cd Student_Analyzer
```

Install pandas

```bash
pip install pandas
```

Run the project

```bash
python main.py
```
