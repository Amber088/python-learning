# 📊 Data Analysis

Hands-on practice and mini-projects covering the core data analysis toolkit — Pandas and NumPy — as part of my GenAI engineering journey.

---

## 🧭 What's Covered

- **Pandas** — DataFrames, filtering, groupby, merging, handling missing data, aggregations
- **NumPy** — arrays, indexing/slicing, broadcasting, vectorized operations, feature engineering

---

Each subfolder contains a notebook/script with working code, plus a short note on what the exercise covers.

---

## 🎯 Why This Matters

Data analysis is the foundation everything else builds on — machine learning, LLM applications, and GenAI systems all start with the ability to clean, transform, and reason about data correctly. These exercises are where that foundation gets built, one concept at a time.

---

## 🔑 Key Concepts Practiced

- Data cleaning and transformation
- Feature engineering (ratios, normalization, derived columns)
- Broadcasting and vectorized operations (no manual loops)
- Aggregation and groupby logic
- Working with real-shaped, messy-ish datasets, not just clean toy examples
